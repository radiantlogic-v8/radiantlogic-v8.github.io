    This class provides a set of methods to perform LDAP client operations on any LDAP server. Note: if you intend to connect to a local FID server, you may prefer to use FIDOperationsClientUtil instead, which has been designed for that purpose.

    Establishing Connections

    Depending on the LDAPOperations2 constructor you use, a connection will or will not be established at the time that new LDAPOperations2 instance is initialized.

    Method #1: Implicit connection during object creation
    The constructors listed below will establish an initial connection when they are called:
    • LDAPOperations2(ConnString2)
    • LDAPOperations2(String)
    • LDAPOperations2(String, String, String)
    • LDAPOperations2(VDQuery)


    Example: Establishing a connection implicitly during creation of the LDAPOperations2 object 
     
         // Note: a connection is established during initialization of the LDAPOperations2 object.
         // The connection is automatically closed when exiting the try-catch block.
         try (LDAPOperations2 ldapOperations2 = new LDAPOperations2("ldap://localhost:2389",
                 "uid=namespaceadmin,ou=globalusers,cn=config", "myPasswordHere")) {
             // TODO: Perform one or more operations using the ldapOperations2 instance.
         } catch (LDAPException e) {
             // TODO: handle or log the exception.
         }

    Method #2: Explicit connection after object creation
    The constructors listed below will NOT establish an initial connection when they are called:
    • LDAPOperations2()
    • LDAPOperations2(Logger)

    You can establish a connection at any time after creating the LDAPOperations2 object, by calling any of the doConnect methods:
    • doConnect(ConnString2)
    • doConnect(String, String, String)
    • doConnect(String, String, String, Vector)
    • doConnect(String, String, String, Vector, boolean, boolean)
    • doConnect(String, String, String, Vector, boolean, boolean, boolean, boolean, String)
    • doConnect(VDQuery)


    Example: Creating an empty LDAPOperations2 instance and establishing a connection explicitly 
     
         LDAPOperations2 ldapOperations2 = new LDAPOperations2();
         try {
            ldapOperations2.doConnect("ldap://localhost:2389",
                     "uid=namespaceadmin,ou=globalusers,cn=config", "myPasswordHere");
            // TODO: perform one or more LDAP operations using ldapOperations2
         } catch (LDAPException e) {
             // TODO: handle or log the exception.
         } finally {
             if (ldapOperations2 != null) {
                 ldapOperations2.doDisconnect();
             }
         }

    Closing Connections

    Once they are no longer used, connections can and should be closed. This can be done explicitly by calling doDisconnect(). A typical example is shown below.

    Example: Creating and closing a connection in a try-catch-finally block 

     
         LDAPOperations2 ldapOperations2 = null;
         try {
             ldapOperations2 = new LDAPOperations2("ldap://localhost:2389",
                     "uid=namespaceadmin,ou=globalusers,cn=config", "myPasswordHere");
         } catch (LDAPException e) {
             // TODO: handle or log the exception.
         } finally {
             if (ldapOperations2 != null) {
                 ldapOperations2.doDisconnect();
             }
         }

    This can also be done implicitly (the doDisconnect() method will be called automatically without having to explicitly call it) by using a try-with-resources statement, as shown below.

    Example: Creating a closing a connection using a try-with-resources block 

     
         // Note: the connection will automatically be closed after exiting the try/catch block.
         try (LDAPOperations2 ldapOperations2 = new LDAPOperations2("ldap://localhost:2389",
                 "uid=namespaceadmin,ou=globalusers,cn=config", "myPasswordHere")) {
             // TODO: Perform one or more operations using the ldapOperations2 instance.

             // Example: base search
             LDAPEntry ldapEntry = ldapOperations2.searchBase("uid=readonly,ou=globalusers,cn=config");

             // Example: delete an entry
             ldapOperations2.delete("uid=Adan_Funston,ou=Accounting,o=companydirectory");
         } catch (LDAPException e) {
             // TODO: handle or log the exception.
         }


    Full Examples

    Example: Connecting to an LDAP server and performing a sub-tree search 
     
         try (LDAPOperations2 ldapOperations2 = new LDAPOperations2("ldap://localhost:2389", "uid=readonly,ou=globalusers,cn=config",
                 "passwordHere")) {
             LDAPEnumeration ldapEnumeration = ldapOperations2.searchWithPaging("o=companydirectory",
                     SearchControls.SUBTREE_SCOPE, "(objectclass=inetorgperson)", new String[]{"cn", "givenName", "mail"}, 10);
             while (ldapEnumeration.hasMore()) {
                 LDAPEntry ldapEntry = ldapEnumeration.next();
                 LDAPAttribute cnAttribute = ldapEntry.getAttribute("cn");
                 Collection<String> cnValues = cnAttribute.getListValueString();
             }
         } catch (LDAPException e) {
             // TODO: handle or log the exception.
         }


    Example: Connecting to an LDAP Server using a pre-defined data source, and adding member values to an existing group 
     
         ConnString2 connectionInfo = null;
         try {
             connectionInfo = DatasourceOperations.getInstance().find("mydatasource");
         } catch (Exception e) {
             // This exception will occur if you have not registered the data source name in your configuration.
             // TODO: handle or log the exception.
         }

         if (connectionInfo != null) {
             try (LDAPOperations2 ldapOperations2 = new LDAPOperations2(connectionInfo)) {
                 String groupDn = "cn=Sales,ou=Groups,o=Test";
                 String newMemberOne = "cn=John Smith,ou=People,o=Test";
                 String newMemberTwo = "cn=Jane Patterson,ou=People,o=Test";
                 LDAPModification addMemberValues = new LDAPModification(LDAPModification.ADD,
                         new LDAPAttribute("uniqueMember", Arrays.asList(newMemberOne, newMemberTwo)));
                 LDAPModification[] modifications = new LDAPModification[]{addMemberValues};
                 ldapOperations2.modify(groupDn, modifications);
             } catch (LDAPException e) {
                 // TODO: handle or log the exception.
             }
         }
